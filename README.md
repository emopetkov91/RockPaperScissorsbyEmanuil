# Rock Paper Scissors Game

Simple console game made with Python.

## Project Description

This is a classic Rock Paper Scissors game where the player competes against the computer.

The game includes:
- Random computer moves
- Best of 3 system
- Colored console text
- Score tracking
- Input validation

## Technologies Used

- Python
- Random module
- Colorama

## How To Play

Choose:
- r for Rock
- p for Paper
- s for Scissors

First to win 2 rounds wins the match.

## Features

- Best of 3 gameplay
- Colored terminal messages
- Scoreboard
- Winner announcement
